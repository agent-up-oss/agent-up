# Security Policy

Agent-Up is a development preview and is not security hardened for production use.

## Reporting a Vulnerability

Do not open public issues for active vulnerability details.

Report security concerns privately to the repository owner through GitHub private vulnerability reporting when available. If private reporting is not enabled, contact the maintainer directly and include enough detail to reproduce the issue.

## Supported Versions

There are no stable supported versions yet. The default branch is the active development line.

## Expectations

The project is maintained on a best-effort basis. Security reports will be reviewed as time allows, but no commercial response time or remediation guarantee is currently offered.
