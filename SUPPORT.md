# Support

Agent-Up is an experimental open-source development preview. It is not commercially supported.

Use GitHub issues for reproducible bugs with clear steps, expected behavior, actual behavior, operating system, .NET SDK version, and relevant logs.

Use GitHub discussions, when enabled, for usage questions, design discussion, and early feedback.

Platform support is limited. The current development setup has been verified on NixOS first; Agent-Up may work elsewhere, but unverified platforms should be treated as experimental.
